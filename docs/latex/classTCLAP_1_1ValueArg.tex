\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class TCLAP\+::\+Value\+Arg$<$ T $>$}

The basic labeled argument that parses a value. 

This is a template class, which means the type T defines the type that a given object will attempt to parse when the flag/name is matched on the command line. While there is nothing stopping you from creating an unflagged \mbox{\hyperlink{classTCLAP_1_1ValueArg}{Value\+Arg}}, it is unwise and would cause significant problems. Instead use an \mbox{\hyperlink{classTCLAP_1_1UnlabeledValueArg}{Unlabeled\+Value\+Arg}}.
