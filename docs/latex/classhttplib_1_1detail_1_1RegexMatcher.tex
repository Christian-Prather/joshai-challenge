\doxysubsection{Detailed Description}
Performs std\+::regex\+\_\+match on request path and stores the result in Request\+::matches. 

Note that regex match is performed directly on the whole request. This means that wildcard patterns may match multiple path segments with /\+: \char`\"{}/begin/(.$\ast$)/end\char`\"{} will match both \char`\"{}/begin/middle/end\char`\"{} and \char`\"{}/begin/1/2/end\char`\"{}.
