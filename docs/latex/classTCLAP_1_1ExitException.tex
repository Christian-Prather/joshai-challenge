\doxysubsection{Detailed Description}
Thrown when TCLAP thinks the program should exit. 

For example after parse error this exception will be thrown (and normally caught). This allows any resource to be clened properly before exit.

If exception handling is disabled (\mbox{\hyperlink{classTCLAP_1_1CmdLine_aa02055d8f4864bfa9b505e2d26bbbd87}{Cmd\+Line\+::set\+Exception\+Handling}}), this exception will propagate to the call site, allowing the program to catch it and avoid program termination, or do it\textquotesingle{}s own cleanup. See for example, \href{https://sourceforge.net/p/tclap/bugs/29}{\texttt{ https\+://sourceforge.\+net/p/tclap/bugs/29}}.
