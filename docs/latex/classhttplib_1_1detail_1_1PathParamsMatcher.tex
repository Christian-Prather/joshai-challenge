\doxysubsection{Detailed Description}
Captures parameters in request path and stores them in Request\+::path\+\_\+params. 

Capture name is a substring of a pattern from \+: to /. The rest of the pattern is matched agains the request path directly Parameters are captured starting from the next character after the end of the last matched static pattern fragment until the next /.

Example pattern\+: \char`\"{}/path/fragments/\+:capture/more/fragments/\+:second\+\_\+capture\char`\"{} Static fragments\+: \char`\"{}/path/fragments/\char`\"{}, \char`\"{}more/fragments/\char`\"{}

Given the following request path\+: \char`\"{}/path/fragments/\+:1/more/fragments/\+:2\char`\"{} the resulting capture will be \{\{\char`\"{}capture\char`\"{}, \char`\"{}1\char`\"{}\}, \{\char`\"{}second\+\_\+capture\char`\"{}, \char`\"{}2\char`\"{}\}\}
